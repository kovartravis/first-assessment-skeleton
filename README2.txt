Channels - A new group can be created with command 'newgroup (groupname)'
            when you create a group you are added to it 
        
           An existing group can be joined with command 'joingroup (groupname)'

           To chat to group use command '@groupname'
            this functions like a whisper but sends message to everyone in group

           Leave a group using the command 'leavegroup (groupname)'

Login - When a new user connects, the server prompts the user for a new password
        When an existing user logs in they will be prompted for their password
        If you are not logged in all commands are ignored by the server
